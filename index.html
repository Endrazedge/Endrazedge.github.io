Tentang Saya

Endra Zulfikar Ghiffari

Halo Indonesia! Saya adalah mahasiswa aktif Program Studi Informatika (Pendidikan Jarak Jauh) di Universitas Siber Asia, dengan minat utama dalam bidang teknologi informasi, khususnya pengembangan website. Ketertarikan saya pada dunia digital mendorong saya untuk terus belajar secara mandiri dan eksploratif, baik melalui kursus daring, forum diskusi, maupun praktik langsung membuat proyek web sederhana. Saya percaya bahwa pemahaman teknologi bukan hanya soal teori, tetapi juga pengalaman dan ketekunan dalam menerapkannya secara nyata.

Selain aktif dalam perkuliahan, saya juga membangun personal branding melalui media sosial, khususnya Instagram, untuk berbagi konten edukatif, tips seputar IT, dan inspirasi produktivitas. Saya senang mempelajari hal baru, adaptif terhadap tren teknologi terkini, serta memiliki semangat untuk tumbuh bersama komunitas digital yang positif. Bagi saya, teknologi adalah jembatan untuk menciptakan solusi nyata dan memberikan dampak bagi banyak orang.
